# Login-Google-with-DB
Login de Google guardando informacion de usuario en BAse de Datos MySql
Crear proyecto en Consola Google Developer
Con el Client Id , el Token y el Redirect URI  configurar el archivo login.php
Crear la base de datos my_db2 en MySql
importar el archivo.SQL
Instalar Composer (recordar tener la version de Xampp con PHP 8.1 o superior)
correr composer require google/apiclient:"^2.19.2"
copiar el codigo y ejecutar login.php
